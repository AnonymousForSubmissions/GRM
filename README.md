# GRM

This is the code for the paper "Adaptive Subgraph Generation for Graph OOD Generalization" submitted to NeurIPS 2023, . 

### Requirement:
torch==1.8.1

### Code Runing:
Run the command: 
`unzip data.zip`

Then run the command:
`python mainl2aug.py`

To switch the dataset, modify the "datasets" value in parse.py.  

The dataset directory should also be specificed in parse.py.
